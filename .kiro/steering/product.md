# Product Overview

The Productivity Dashboard is a client-side web application designed to help users manage their time and tasks effectively. It combines essential productivity tools in a single, clean interface without requiring any backend infrastructure or user authentication.

## Core Features

- **Time & Date Display**: Real-time clock with time-based greetings (Morning/Afternoon/Evening/Night)
- **Focus Timer**: 25-minute Pomodoro-style countdown timer for focused work sessions
- **Task Management**: Full-featured to-do list with create, edit, delete, and completion tracking
- **Quick Links**: Customizable shortcuts to frequently visited websites

## Key Characteristics

- **Zero Backend**: Entirely client-side application using browser Local Storage for persistence
- **Privacy-First**: All data stays in the user's browser, no server communication
- **Instant Access**: No login, no setup - just open and start using
- **Offline-Capable**: Works without internet connection once loaded

## Target Use Case

Ideal for users who want a simple, distraction-free productivity dashboard that lives in their browser. Perfect for students, remote workers, and anyone practicing time management techniques like Pomodoro.
